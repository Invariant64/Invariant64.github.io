Tech Otakus Save The World

发表于 2022-09-05 更新于 2022-09-12 阅读次数：

About

Tech otakus save the world这句话，最初是在miHoYo的一款游戏《崩坏3》看到的（貌似每个miHoYo的游戏加载界面都有这句话），那时年龄比较小，也不清楚otaku的具体含义，便没有过多在意。

前段时间，在网络上看见了miHoYo的入职说明书，这句话出现在了说明书的最开头（这句话貌似是miHoYo的公司精神一类的？）。

直到前几日完善Github上的个人页面时，有一栏是自己的简介，这时我想起这句话来，发现这句话还挺适合，便打了上去，后来也用在了自己的博客简介上。

说明一下，虽然我以前也玩过miHoYo的游戏，也很喜欢其中的一些角色，但是，我认可这句话和miHoYo本身关系不大，和他们的IP或是游戏也没有关系，所以我个人对这句话的理解可能和写出这句话的人的理解相差甚远，在这里只是记录下我的一家之言。

Tech

Tech不必多说，代表了technology，即技术。

技术带给世界的变化大家有目共睹，在科学技术高速发展的21世纪，技术迭代的速度远超我们的想象，尤其是近20年，技术使人类世界发生了翻天覆地的变化。

Otakus

Otaku这个词源自日本，原指的是御宅族一类的人。

广义的御宅族指热衷于各种亚文化，并对该文化有极度深入的了解的人。 比如“体育宅”、“音乐宅”、“汽车宅”、“铁路宅”、“军事宅”、“枪械宅”、“摄影宅”、“化学宅”、“植物宅”‘、“方言宅”等等。
狭义的御宅族指沉溺、热衷或博精于动画、漫画以及电子游戏的人。

——百度百科

显然，这句话中的otakus指的是广义的御宅族们。从我个人理解的角度来说，这里的Tech otakus应该指的就是技术宅。

技术宅指在所属亚文化领域中具有较强技术能力的御宅族，这种称谓与ACG宅、铁道宅、军事宅等是不同的分类方式，与以上各种“宅”是交叉存在，而非平行存在。

——同上百度百科

技术宅与一般的拥有技术的人们不同。从字面上看，技术宅也是御宅族，社交属性应该更偏向于网络社交，但是相比于单纯的御宅族，技术宅的社恐属性减少了，也完全不是传统的家里蹲。一般的御宅族之所以家里蹲，是因为游戏、漫画、动漫等宅属性的事情占据了御宅族的精神世界，在这些能带给他们很大的满足的情况下，如非必要，他们不想也无暇外出进行社交活动。（窝在家里真的很舒服！）

而技术宅和御宅族一样，也喜欢窝在家里，不过与御宅族窝在家里打游戏看漫画不同，技术宅在家中的空闲时间会有部分（甚至是大部分）会拿来钻研自己感兴趣的技术，或是进行一些个人的开发工作。之所以技术宅会被冠以“宅”的名号，是因为如今网络上被称为“技术宅”的人，很大一部分都是从事电子技术、信息技术相关专业的人，或是相关专业的学生。而这些专业具有一个相同的特点，即相关专业工作的完成对设备的要求很低，仅使用简单的设备就能完成。如计算机相关的工作很多仅使用个人电脑就能完成，即使需要使用到大型服务器一类的设备，也可以通过网络进行调用。所以既然能在家里就能舒服地完成工作，那么何乐而不为呢？

近几年在网络上有一种趋势，技术博客的风格逐渐年轻化，同时亚文化的含量逐渐上升。具体举个例子：在网络上顶着ACGN头像的人热烈地讨论着技术的人越来越多了（笑），这一现象在我国的网络上尤为明显。这说明了亚文化圈与技术圈的交集在不断扩大，年轻人逐渐成为技术领域的中流砥柱。长江后浪推前浪,前浪死在沙滩上技术更新迭代的速度快，年轻人学习能力强，接受新技术的速度快，所以技术领域逐渐年轻化十分正常。而之所以亚文化圈与技术圈的交集扩大，是因为如今绝大多数亚文化内容都依赖互联网平台进行传播，那么喜爱亚文化的人自然对互联网十分熟悉，接触到信息技术内容的几率比一般人更高，也会有更多的人萌发起兴趣，于是两个圈子的交集便扩大了。

Tech & Otaku

我们在评判一项技术的时候，需要同时考虑技术的适用场景，技术处理某件问题的效率，支出与产出的比例等等因素，然后得出类似“这项技术很先进，适合投入生产”或是“这项技术已经过时了”等等结论。一项技术可能在A领域无用武之地，但在B领域就能大放异彩。所以，技术的意义是由人赋予的，独立地评判一项技术的好坏是没有意义的。

而赋予技术意义的人，正是这些技术宅们。

技术宅比其他任何人都要理解技术，而同时他们又是最贴近生活，最能从一名用户的角度对技术进行思考的人，所以“技术”和“宅”是密不可分的。

在我看来，是“宅”赋予了技术以精神内核，这种精神内核是一种类似“灵魂”的东西，有了“灵魂”，技术才有了意义。

正如同一个个游离的汉字本身没有意义，而作家将汉字以自己的方式排列组合，使之构成一篇篇美妙的诗歌、小说······最终形成了可以被称为艺术的东西。当然，技术是否与诗歌一样具有艺术性仍值得商榷，这里只是举了一个不那么恰当的例子，来展示是人赋予了这些无机事物以意义。

The Open Source Spirit

我个人认为，互联网的开源精神和武侠小说中的“侠客精神”有很多相似之处，但其中最重要的一点是：自由。

我有时候会想，技术宅即是武侠小说中的侠客，以自己的能力帮助力所能及的人，自由而不受限制。这虽然只是幼稚的幻想，但现实中确实存在向着这个方向努力的人。比如为开源社区做出贡献的人，他们中的每一个人都值得称赞。

相比宽泛的侠客精神，开源精神具有准确的定义，历史，以及相关的规范，不过这都是题外话了。

End
